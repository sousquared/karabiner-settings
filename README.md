
## complex_modificationsの設定方法
`~/.config/karabiner/assets/complex_modifications/`に置いて、`Cpmplex Modifications→Add predefined rule`からenable allする.

もしくは`Cpmplex Modifications→Add your own rule`に直接コピペする.
